#  Calculator App (Android + React)

A simple and efficient **Calculator Application** built using **Java (Android)** and **React**.  
The app performs basic arithmetic operations with a clean UI and smooth user experience.

---

## Features

-  Addition  
-  Subtraction  
-  Multiplication  
-  Division  
-  Clear / Reset functionality  
-  Fast and responsive performance  
-  User-friendly interface  

---

##  Tech Stack

- **Java** – Android logic and backend  
- **React** – UI / component-based logic  
- **Android Studio** – Development environment  
- **XML** – UI design (Android layouts)  
- **Gradle** – Build system  

---

CalculatorApp/

## 🚀 Installation & Setup

### 1. Clone the repository
```bash
git clone https://github.com/your-username/calculator-app.git
cd calculator-app
```
### 2. Install dependencies
npm install

### 3. Run the app
npm run android
